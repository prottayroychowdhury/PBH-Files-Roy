# PBH Accretion

This repository contains my current work on extending the primordial black hole (PBH) accretion model of Ali-Haïmoud & Kamionkowski (2017), with the goal of improving predictions for the resulting CMB constraints on PBH abundance.

The code includes reproductions and extensions of the steady-state accretion model, alongside ongoing work exploring the expansion-dominated regime for higher masses.

## Recombination History

`output_xe.dat` contains the background ionization and matter temperature history used by the accretion calculations. It was generated using **HYREC-2**:

https://github.com/nanoomlee/HYREC-2
